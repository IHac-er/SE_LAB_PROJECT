What are HTML Lists?

Lists in HTML help organize content into ordered, unordered, and definition lists.

Unordered Lists (<ul>)

Unordered lists use bullet points.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered Lists (<ol>)

Ordered lists use numbers.

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
    <li>Step 3</li>
</ol>

Definition Lists (<dl>)

Definition lists include terms and their descriptions.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

Nested Lists

Lists can be nested within other lists.

<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrot</li>
            <li>Broccoli</li>
        </ul>
    </li>
</ul>
