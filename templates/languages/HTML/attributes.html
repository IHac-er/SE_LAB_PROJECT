What are HTML Attributes?

Attributes provide additional information about an HTML element.

Common HTML Attributes

  • href - Specifies the URL for links.
  • src - Specifies the source for images, videos, or scripts.
  • alt - Provides alternative text for images.
  • id - Unique identifier for an element.
  • class - Assigns a CSS class to an element.
  • style - Inline CSS styling.
  • title - Tooltip text when hovering over an element.
  • disabled - Disables an input field or button.

Example: Using Attributes

<a href="https://www.example.com" title="Visit Example">Click Here</a>

<img src="image.jpg" alt="A sample image">

<p id="unique" class="text-bold" style="color: blue;">This is a paragraph.</p>

Global Attributes

These attributes can be applied to any HTML element:

  • id - Unique identifier.
  • class - Specifies CSS class.
  • style - Inline styles.
  • title - Tooltip text.
  • hidden - Hides the element.
