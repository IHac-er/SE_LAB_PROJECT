What are HTML Tables?

HTML tables allow you to organize data into rows and columns.

Basic Table Structure

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
        <td>USA</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
        <td>UK</td>
    </tr>
</table>

Table Elements

  • <table> - Defines the table.
  • <tr> - Table row.
  • <th> - Table header cell.
  • <td> - Table data cell.
  • <caption> - Adds a title to the table.
  • <thead>, <tbody>, <tfoot> - Organizes table content.

Advanced Table Features

<table border="1">
    <caption>Student Scores</caption>
    <thead>
        <tr>
            <th>Name</th>
            <th>Math</th>
            <th>Science</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Charlie</td>
            <td>85</td>
            <td>90</td>
        </tr>
        <tr>
            <td>Diana</td>
            <td>92</td>
            <td>88</td>
        </tr>
    </tbody>
</table>
