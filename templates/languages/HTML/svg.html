What is SVG?

SVG (Scalable Vector Graphics) is an XML-based format for vector graphics.

Basic SVG Example

SVG allows you to create vector-based graphics with shapes like circles, rectangles, and paths.

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" stroke="black" stroke-width="3" fill="blue" />
</svg>

SVG Elements

  • <circle> - Draws a circle.
  • <rect> - Draws a rectangle.
  • <line> - Draws a straight line.
  • <polygon> - Draws a polygon.
  • <path> - Defines custom paths.

Example: Rectangle and Line

<svg width="300" height="200">
    <rect width="200" height="100" x="50" y="50" fill="green" stroke="black" stroke-width="3"/>
    <line x1="50" y1="50" x2="250" y2="150" stroke="red" stroke-width="3"/>
</svg>
