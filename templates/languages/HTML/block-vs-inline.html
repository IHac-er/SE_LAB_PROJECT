What are Block and Inline Elements?

HTML elements are categorized into two types: block-level elements and inline elements.

Block Elements

Block elements take up the full width available and start on a new line.

Examples of block elements:

<div>This is a block element</div>
<p>Paragraphs are block elements</p>
<h1>Headings are block elements</h1>

Inline Elements

Inline elements take up only as much width as necessary and do not start on a new line.

Examples of inline elements:

<span>This is an inline element</span>
<a href="#">Links are inline</a>
<strong>Bold text is inline</strong>

Key Differences

Block Elements Inline Elements
Start on a new line Do not start on a new line
Take up full width Take only necessary width
Can contain other block and inline elements Can contain only text or other inline elements
