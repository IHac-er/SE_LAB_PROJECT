What are Pointers?

Pointers in C are variables that store the memory address of another variable.

Why Use Pointers?

  • Efficient memory management.
  • Allows dynamic memory allocation.
  • Enables efficient array and string handling.
  • Used for function arguments to modify actual values.

Declaring and Using Pointers

#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = # // Pointer storing address of num

    printf("Value of num: %d\n", num);
    printf("Address of num: %p\n", &num);
    printf("Pointer pointing to: %p\n", ptr);
    printf("Value via pointer: %d\n", *ptr);
    
    return 0;
}

Pointer Arithmetic

Pointers can be incremented or decremented to traverse memory locations.

#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30};
    int *ptr = arr;

    printf("First element: %d\n", *ptr);
    ptr++;
    printf("Second element: %d\n", *ptr);
    ptr++;
    printf("Third element: %d\n", *ptr);
    
    return 0;
}

Pointers and Arrays

Arrays and pointers are closely related in C.

#include <stdio.h>

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int *ptr = arr;

    for (int i = 0; i < 5; i++) {
        printf("%d ", *(ptr + i));
    }

    return 0;
}

Pointers and Functions

Pointers can be used to modify values in functions.

#include <stdio.h>

void modify(int *ptr) {
    *ptr = 100;
}

int main() {
    int num = 10;
    modify(&num);
    printf("Modified value: %d\n", num);
    
    return 0;
}

Dynamic Memory Allocation

Pointers are used for dynamic memory allocation.

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr = (int*) malloc(sizeof(int));

    if (ptr == NULL) {
        printf("Memory allocation failed.\n");
        return 1;
    }

    *ptr = 50;
    printf("Dynamically allocated value: %d\n", *ptr);

    free(ptr);
    return 0;
}
