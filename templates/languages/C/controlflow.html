Understanding Control Flow in C

Control flow determines the execution order of statements in a C program.

What is Control Flow?

Control flow in C refers to the execution order of statements based on conditions and loops.

Decision-Making Statements

  • if Statement: Executes a block of code if a condition is true.
  • if-else Statement: Executes one block if true, another if false.
  • switch Statement: Used when multiple conditions depend on a single variable.

Example: if-else Statement

#include <stdio.h>

int main() {
    int num = -5;

    if (num > 0) {
        printf("Positive number\n");
    } else {
        printf("Negative number\n");
    }

    return 0;
}

Looping Statements

  • for Loop: Repeats a block of code a fixed number of times.
  • while Loop: Executes as long as the condition is true.
  • do-while Loop: Executes at least once, then checks condition.

Example: for Loop

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }

    return 0;
}

Break and Continue Statements

  • break Statement: Exits a loop or switch statement immediately.
  • continue Statement: Skips the current iteration and continues with the next.

Example: break Statement

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break;
        }
        printf("%d\n", i);
    }

    return 0;
}
