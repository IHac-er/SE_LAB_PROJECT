What are Preprocessor Directives?

Preprocessor directives in C are commands that are processed before compilation begins. They start with a `#` symbol.

Types of Preprocessor Directives

  • File Inclusion: `#include`
  • Macro Definition: `#define`
  • Conditional Compilation: `#ifdef`, `#ifndef`, `#if`, `#else`, `#elif`, `#endif`
  • Other Directives: `#pragma`, `#undef`

1. File Inclusion (`#include`)

Includes external header files in a C program.

#include <stdio.h>
#include "myfile.h"  // User-defined header file

2. Macro Definition (`#define`)

Used to define constants or macros.

#define PI 3.1416

int main() {
    printf("Value of PI: %f\n", PI);
    return 0;
}

3. Conditional Compilation

Controls whether certain parts of the code should be compiled.

#include <stdio.h>

#define DEBUG 1

int main() {
#ifdef DEBUG
    printf("Debugging mode is ON\n");
#endif
    return 0;
}

4. `#pragma` Directive

Used for compiler-specific instructions.

#pragma startup myFunction
#pragma exit myFunction
