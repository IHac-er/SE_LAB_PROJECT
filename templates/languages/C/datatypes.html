What are Data Types in C?

Data types in C define the type of data that a variable can store.

Types of Data Types in C

  • Primary Data Types: int, char, float, double.
  • Derived Data Types: Arrays, Pointers, Structures, Unions.
  • User-Defined Data Types: typedef, enum.

Size of Data Types

  • int: Typically 4 bytes.
  • char: 1 byte.
  • float: 4 bytes.
  • double: 8 bytes.

Example: Declaring Data Types

#include <stdio.h>

int main() {
    int num = 10;
    char letter = 'A';
    float pi = 3.14;
    double largeDecimal = 3.1415926535;

    printf("Integer: %d\n", num);
    printf("Character: %c\n", letter);
    printf("Float: %f\n", pi);
    printf("Double: %lf\n", largeDecimal);

    return 0;
}

Signed and Unsigned Data Types

  • Signed int: Can store both positive and negative values.
  • Unsigned int: Can store only positive values, but with a higher range.
  • Signed char: Can store both negative and positive characters.
  • Unsigned char: Can store only positive characters.

Example: Signed vs Unsigned

#include <stdio.h>

int main() {
    signed int a = -5;
    unsigned int b = 5;

    printf("Signed int: %d\n", a);
    printf("Unsigned int: %u\n", b);

    return 0;
}

Derived Data Types

  • Arrays: Collection of elements of the same data type.
  • Pointers: Stores the memory address of another variable.
  • Structures: Groups related variables of different data types.
  • Unions: Similar to structures but shares memory among members.

Example: Using Arrays

#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }

    return 0;
}

User-Defined Data Types

  • typedef: Used to define a new data type.
  • enum: Used to define a set of named integer constants.

Example: Using typedef

#include <stdio.h>

typedef unsigned int uint;

int main() {
    uint age = 25;
    printf("Age: %u\n", age);

    return 0;
}

Example: Using enum

#include <stdio.h>

enum Weekday {Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday};

int main() {
    enum Weekday today = Wednesday;
    printf("Today is day number: %d\n", today);

    return 0;
}
