CSS Selectors

Learn how to target HTML elements using CSS selectors.

What are CSS Selectors?

CSS selectors define the elements to which CSS rules apply.

Basic Selectors

  • Element Selector: Targets elements by tag name.
  • ID Selector (#id): Targets an element with a specific ID.
  • Class Selector (.class): Targets elements with a specific class.
  • Universal Selector (*): Targets all elements.

/* Example */
p {
    color: blue;
}

#unique {
    font-size: 20px;
}

.highlight {
    background-color: yellow;
}

* {
    margin: 0;
    padding: 0;
}

Group Selectors

Multiple selectors can be grouped together using a comma:


h1, h2, p {
    color: white;
}

Combinators

  • Descendant Selector (A B): Targets B inside A.
  • Child Selector (A > B): Targets direct children B inside A.
  • Adjacent Sibling (A + B): Targets the next sibling B of A.
  • General Sibling (A ~ B): Targets all siblings B of A.

/* Example */
div p {
    color: green;
}

div > p {
    font-weight: bold;
}

h1 + p {
    font-style: italic;
}

h1 ~ p {
    text-decoration: underline;
}

Pseudo-Classes and Pseudo-Elements

Pseudo-classes and pseudo-elements allow advanced styling.


/* Pseudo-Class */
a:hover {
    color: red;
}

/* Pseudo-Element */
p::first-letter {
    font-size: 2em;
}
