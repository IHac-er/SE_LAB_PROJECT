CSS Positioning

CSS Positioning is used to control the placement of elements on a webpage.

What is CSS Positioning?

CSS provides different ways to position HTML elements on a webpage. The position property allows you to define how elements appear relative to the document flow or other elements.

Types of Positioning

There are five main types of positioning in CSS:

  • Static - Default positioning; follows normal document flow.
  • Relative - Positioned relative to its original position.
  • Absolute - Positioned relative to the nearest positioned ancestor.
  • Fixed - Positioned relative to the viewport; does not move when scrolling.
  • Sticky - Switches between relative and fixed positioning depending on scroll position.

Static Positioning

By default, all elements are positioned as static. They appear in the normal document flow.

<div class="static">This is a statically positioned div</div>

Relative Positioning

When an element has position: relative;, it moves relative to its normal position.

<div class="relative">This is a relatively positioned div</div>

Absolute Positioning

With position: absolute;, an element is positioned relative to the nearest positioned ancestor (excluding static elements).

<div class="absolute">This is an absolutely positioned div</div>

Fixed Positioning

Elements with position: fixed; remain fixed relative to the viewport and do not scroll.

<div class="fixed">This is a fixed positioned div</div>

Sticky Positioning

position: sticky; makes an element act like relative until a certain scroll position is reached, then it behaves like fixed.

<div class="sticky">This is a sticky positioned div</div>
