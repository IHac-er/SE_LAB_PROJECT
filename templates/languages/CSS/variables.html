CSS Variables

Learn how to use variables in CSS to create reusable and flexible styles.

What are CSS Variables?

CSS Variables (also known as custom properties) allow you to store values in reusable variables and apply them throughout your stylesheet.

Basic Syntax

:root {
    --primary-color: #ffcc00;
    --secondary-color: #333;
    --font-size: 16px;
}

body {
    background-color: var(--secondary-color);
    color: var(--primary-color);
    font-size: var(--font-size);
}

Defining and Using Variables

1. Declaring Variables

CSS variables are typically defined inside the `:root` selector to make them global.

:root {
    --main-bg-color: black;
    --text-color: white;
}

2. Applying Variables

Use the `var()` function to access the variable values.

body {
    background-color: var(--main-bg-color);
    color: var(--text-color);
}

3. Variable Scope

Variables declared inside a selector are scoped to that selector.

.container {
    --container-bg: blue;
    background-color: var(--container-bg);
}

Advantages of CSS Variables

  • Improves maintainability of stylesheets.
  • Allows easy theme changes.
  • Reduces repetition of values.
  • Supports real-time changes via JavaScript.

CSS Variables with JavaScript

CSS variables can be dynamically updated using JavaScript.

document.documentElement.style.setProperty('--primary-color', 'blue');
