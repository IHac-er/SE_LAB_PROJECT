Introduction to CSS

Understanding the fundamentals of Cascading Style Sheets (CSS).

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation of web pages. It allows you to define styles such as colors, layouts, fonts, and spacing.

Why Use CSS?

  • Enhances the appearance of web pages.
  • Separates content from design.
  • Improves website performance and user experience.
  • Allows for responsive and adaptive design.

How CSS Works

CSS is applied to HTML elements using different methods:

  • Inline CSS: Applied directly to an HTML element using the style attribute.
  • Internal CSS: Defined inside a <style> tag in the HTML file.
  • External CSS: Stored in a separate .css file and linked using the <link> tag.

Example: Basic CSS

The following example changes the background color and text color of an HTML page.


/* styles.css */
body {
    background-color: black;
    color: white;
    font-family: Arial, sans-serif;
}
