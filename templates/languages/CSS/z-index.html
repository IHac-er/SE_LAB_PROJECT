CSS Z-Index

The z-index property controls the vertical stacking order of elements.

What is Z-Index?

The z-index property defines the stack order of elements. Elements with a higher z-index appear in front of elements with a lower z-index.

It only works on elements with a position of relative, absolute, fixed, or sticky.

How Z-Index Works

The default stack order follows the document flow, but you can control it using z-index.

<div class="box box1">Box 1 (z-index: 1)</div>
<div class="box box2">Box 2 (z-index: 2)</div>

Example of Z-Index

In the example below, Box 2 will appear on top of Box 1 because it has a higher z-index.

.box1 {
    position: absolute;
    top: 50px;
    left: 50px;
    width: 100px;
    height: 100px;
    background-color: red;
    z-index: 1;
}

.box2 {
    position: absolute;
    top: 70px;
    left: 70px;
    width: 100px;
    height: 100px;
    background-color: blue;
    z-index: 2;
}

Auto and Negative Z-Index

The default z-index value is "auto," which means the stacking order follows the document flow.

Negative values push elements behind others.

.box3 {
    position: absolute;
    top: 90px;
    left: 90px;
    width: 100px;
    height: 100px;
    background-color: green;
    z-index: -1;
}
