CSS Pseudo-Classes

Pseudo-classes define the special state of an element based on user interaction or document structure.

What are Pseudo-Classes?

Pseudo-classes are keywords added to selectors that define a special state of the selected elements. They allow styling elements dynamically without adding extra classes or JavaScript.

Basic Syntax

selector:pseudo-class {
    /* CSS properties */
}

Common Pseudo-Classes

1. Hover Effect

Changes the background color of a button when hovered over.

button:hover {
    background-color: yellow;
}

2. Focus State

Highlights an input field when it is focused.

input:focus {
    border: 2px solid blue;
}

3. Active State

Changes the appearance of a link when it is being clicked.

a:active {
    color: red;
}

4. First Child and Last Child

Applies styles to the first and last list item.

li:first-child {
    font-weight: bold;
}

li:last-child {
    font-style: italic;
}

5. nth-child Selector

Targets the third child in a list.

li:nth-child(3) {
    color: green;
}

6. Checked State

Styles checkboxes when selected.

input[type="checkbox"]:checked {
    outline: 2px solid green;
}

7. Disabled and Enabled States

Styles buttons based on their disabled state.

button:disabled {
    background-color: gray;
    cursor: not-allowed;
}

Combining Multiple Pseudo-Classes

You can use multiple pseudo-classes together for advanced styling.

Example: Styling every odd list item except the first one

li:nth-child(odd):not(:first-child) {
    background-color: lightgray;
}
