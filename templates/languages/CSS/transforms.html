CSS Transforms

CSS Transforms allow elements to be rotated, scaled, skewed, and translated in 2D or 3D space.

2D Transforms

2D transforms allow modifying an elementâ€™s appearance in a two-dimensional space.

Translate

The `translate(x, y)` function moves an element from its current position.

.box {
    transform: translate(50px, 20px);
}

Rotate

The `rotate(deg)` function rotates an element by a specified degree.

.box {
    transform: rotate(45deg);
}

Scale

The `scale(x, y)` function resizes an element.

.box {
    transform: scale(1.5, 1.5);
}

Skew

The `skew(x, y)` function skews an element along the X and Y axes.

.box {
    transform: skew(20deg, 10deg);
}

3D Transforms

3D transforms add depth to elements by modifying their position in a three-dimensional space.

RotateX, RotateY, RotateZ

These functions rotate an element along different axes.

.box {
    transform: rotateX(45deg);
}

Perspective

The `perspective(value)` function adds depth by determining how far an object appears.

.container {
    perspective: 500px;
}

.box {
    transform: rotateY(45deg);
}

Combining Multiple Transformations

Multiple transformations can be applied in one `transform` property.

.box {
    transform: rotate(45deg) scale(1.2) translate(20px, 30px);
}
