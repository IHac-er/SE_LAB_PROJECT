CSS Box Model

Understanding how elements are structured and spaced.

What is the Box Model?

Every HTML element is a rectangular box. The CSS Box Model consists of:

  • Content: The actual text or images inside the box.
  • Padding: Space between content and border.
  • Border: Surrounds the padding and content.
  • Margin: Space outside the border, separating elements.

Example: Box Model in Action


.box {
    width: 200px;
    padding: 20px;
    border: 5px solid #ffcc00;
    margin: 10px;
    background-color: #333;
    color: white;
}

This creates a box with content, padding, border, and margin.
