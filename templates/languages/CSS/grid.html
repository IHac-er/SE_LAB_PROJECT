CSS Grid

CSS Grid is a two-dimensional layout system that allows precise control over rows and columns.

What is CSS Grid?

CSS Grid is a powerful layout system that allows web developers to create complex layouts using rows and columns easily.

Basic Grid Properties

  • display: grid; - Enables grid layout.
  • grid-template-columns - Defines column structure.
  • grid-template-rows - Defines row structure.
  • grid-gap - Adds space between grid items.
  • justify-items - Aligns items horizontally.
  • align-items - Aligns items vertically.

Example: Creating a Grid Layout


/* Grid Container */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 100px);
    gap: 10px;
    background-color: #333;
    padding: 10px;
}

.grid-item {
    background-color: #ffcc00;
    text-align: center;
    line-height: 100px;
    color: black;
    font-weight: bold;
}

Grid Example in HTML


<div class="grid-container">
