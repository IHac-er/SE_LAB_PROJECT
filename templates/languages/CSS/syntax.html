CSS Syntax

Understanding how CSS rules are structured.

Basic CSS Syntax

CSS follows a simple structure where you define rules for HTML elements.


/* CSS Syntax */
selector {
    property: value;
}

Example: Changing Text Color

This example sets the text color of all paragraphs to blue:


p {
    color: blue;
}

Selectors, Properties, and Values

  • Selector: Defines which HTML elements the rule applies to.
  • Property: The aspect of the element you want to style (e.g., color, font-size).
  • Value: The specific setting for the property.

Multiple Properties

CSS allows multiple properties within one selector:


h1 {
    color: white;
    background-color: black;
    text-align: center;
}
