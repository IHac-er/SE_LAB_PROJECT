CSS Media Queries

Media Queries allow the application of CSS rules based on device characteristics such as width, height, resolution, and orientation.

What are Media Queries?

Media queries are used to create responsive designs by applying styles conditionally based on the user's device properties.

Basic Syntax

@media (condition) {
    /* CSS rules here */
}

Common Media Query Examples

1. Adjusting Styles for Small Screens

Applies styles when the screen width is 600px or less.

@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}

2. Applying Styles for Tablets

Targets devices with widths between 600px and 1024px.

@media (min-width: 600px) and (max-width: 1024px) {
    body {
        background-color: lightblue;
    }
}

3. Changing Layout for Large Screens

Applies styles when the screen width is larger than 1024px.

@media (min-width: 1025px) {
    body {
        background-color: white;
    }
}

4. Orientation-Based Media Queries

Styles applied when the device is in landscape mode.

@media (orientation: landscape) {
    body {
        background-color: yellow;
    }
}

5. Retina Display Queries

Applies styles for high-resolution (Retina) screens.

@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
    body {
        background-color: pink;
    }
}

Combining Multiple Conditions

Multiple conditions can be combined using logical operators.

Example: Dark Mode for Mobiles

@media (max-width: 768px) and (prefers-color-scheme: dark) {
    body {
        background-color: black;
        color: white;
    }
}
