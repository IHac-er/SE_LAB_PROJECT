CSS Transitions & Animations

Transitions and animations bring life to web pages by adding smooth effects and movements.

CSS Transitions

CSS transitions allow elements to change property values smoothly over a specified duration.

/* Example: Smooth background color change */
.box {
    width: 100px;
    height: 100px;
    background-color: red;
    transition: background-color 0.5s ease-in-out;
}

.box:hover {
    background-color: blue;
}

CSS Animations

Unlike transitions, CSS animations allow more control over movement with keyframes.

/* Example: Bouncing box animation */
@keyframes bounce {
    0% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
    100% { transform: translateY(0); }
}

.animated-box {
    width: 100px;
    height: 100px;
    background-color: green;
    animation: bounce 1s infinite;
}

Keyframes in CSS

Keyframes define intermediate steps of an animation.

@keyframes fadeIn {
    0% { opacity: 0; }
    100% { opacity: 1; }
}

.fading-box {
    animation: fadeIn 2s ease-in;
}

Difference Between Transitions & Animations

Feature Transitions Animations
Trigger User interaction (hover, click, etc.) Runs automatically
Control Limited control Fully controlled with keyframes
Looping No Yes (can be infinite)
