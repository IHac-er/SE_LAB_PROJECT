Python Syntax Overview

Python is a high-level, interpreted programming language known for its simplicity and readability.

Indentation in Python

Python uses indentation instead of braces `{}` to define code blocks.


# Correct indentation
if True:
    print("This is indented correctly")

# Incorrect indentation (causes an error)
if True:
print("This will cause an IndentationError")

Comments in Python

Python uses `#` for single-line comments and triple quotes for multi-line comments.


# This is a single-line comment

"""
This is a multi-line comment.
It spans multiple lines.
"""

Variables & Assignments

Python does not require variable declarations.


x = 10  # Assigning an integer
name = "Alice"  # Assigning a string
pi = 3.14  # Assigning a float

print(x, name, pi)

Printing Output

The `print()` function is used to display output.


print("Hello, Python!")  # Output: Hello, Python!

Taking User Input

The `input()` function is used to get user input.


name = input("Enter your name: ")
print("Hello, " + name + "!")

Basic Data Types

Python supports various data types, including integers, floats, strings, and booleans.


x = 5       # Integer
y = 3.14    # Float
name = "Python"  # String
is_valid = True  # Boolean

print(type(x), type(y), type(name), type(is_valid))

Simple If-Else Statement

Python uses `if`, `elif`, and `else` for decision-making.


x = 10

if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")

Defining Functions

Functions in Python are created using the `def` keyword.


def greet(name):
    return "Hello, " + name + "!"

print(greet("Alice"))  # Output: Hello, Alice!

Exception Handling

Python uses `try-except` blocks to handle errors.


try:
    num = int(input("Enter a number: "))
    print(10 / num)
except ZeroDivisionError:
    print("You can't divide by zero!")
except ValueError:
    print("Invalid input! Please enter a number.")

Importing Modules

Python allows importing built-in and custom modules.


import math

print(math.sqrt(16))  # Output: 4.0
