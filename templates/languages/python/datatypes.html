Python Data Types

Learn about the different data types in Python.

Common Data Types

  • int - Integer (e.g., 10)
  • float - Floating-point number (e.g., 10.5)
  • str - String (e.g., "Hello")
  • list - List (e.g., [1, 2, 3])
  • tuple - Tuple (e.g., (1, 2, 3))
  • dict - Dictionary (e.g., {"key": "value"})
