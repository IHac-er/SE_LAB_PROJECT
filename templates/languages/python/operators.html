Python Operators

What are Operators?

Operators are special symbols in Python that perform operations on variables and values. Python provides different types of operators to handle **mathematical, logical, bitwise, comparison, and membership** operations.

1. Arithmetic Operators

These operators are used for mathematical calculations.


a = 10
b = 3

print(a + b)  # Addition: 13
print(a - b)  # Subtraction: 7
print(a * b)  # Multiplication: 30
print(a / b)  # Division: 3.333
print(a % b)  # Modulus: 1
print(a ** b) # Exponentiation: 1000 (10Â³)
print(a // b) # Floor division: 3 (rounds down)
