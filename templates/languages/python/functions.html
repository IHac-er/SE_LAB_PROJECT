Python Functions

Learn how to define and use functions in Python, including arguments, return values, recursion, and more.

What are Functions?

A function is a block of reusable code that performs a specific task.

1. Defining and Calling Functions

Functions are defined using the def keyword.


def greet():
    print("Hello, welcome to Python!")

greet()  # Calling the function

2. Function Arguments

Functions can take parameters as input.


def greet(name):
    print("Hello,", name)

greet("Alice")
greet("Bob")

3. Return Values

Functions can return values using the return statement.


def add(a, b):
    return a + b

result = add(3, 5)
print("Sum:", result)

4. Default and Keyword Arguments

Arguments can have default values.


def greet(name="Guest"):
    print("Hello,", name)

greet()  # Uses default value
greet("Alice")

5. Variable-Length Arguments (*args and **kwargs)

*args allows multiple positional arguments, and **kwargs allows multiple keyword arguments.


def sum_numbers(*args):
    return sum(args)

print(sum_numbers(1, 2, 3, 4, 5))

def show_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

show_info(name="Alice", age=25, city="New York")

6. Lambda Functions

Lambda functions are anonymous one-line functions.


square = lambda x: x ** 2
print(square(5))  # Output: 25

add = lambda a, b: a + b
print(add(3, 7))  # Output: 10

7. Recursion

A function can call itself (recursive function).


def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Output: 120
