Python Generators

Learn how to use generators in Python for efficient memory management and iteration.

What are Generators?

Generators in Python allow you to iterate over data without storing it in memory, making them efficient for large datasets.

1. Creating a Generator

Generators use the yield statement instead of return. Each call to next() resumes execution from the last yield.


def count_down(n):
    while n > 0:
        yield n
        n -= 1

gen = count_down(5)
print(next(gen))  # 5
print(next(gen))  # 4

2. Iterating Over a Generator

You can iterate over a generator using a for loop.


def count_up(n):
    for i in range(n):
        yield i

for num in count_up(5):
    print(num)

3. Generator Expressions

Similar to list comprehensions, but using parentheses instead of square brackets.


squares = (x**2 for x in range(5))

for num in squares:
    print(num)

4. Difference Between Generators and Normal Functions

Generators yield values one at a time, while normal functions return values and exit.


# Normal function
def get_numbers():
    return [1, 2, 3]

# Generator function
def get_numbers_gen():
    yield 1
    yield 2
    yield 3

print(get_numbers())     # Returns entire list
print(list(get_numbers_gen()))  # Iterates through generator
