Python Regular Expressions

What are Regular Expressions?

Regular expressions (regex) are powerful tools for **searching, matching, and manipulating** text patterns. Python provides the re module to handle regex efficiently.

1. Importing the re Module

To use regular expressions, first import the re module:

import re

2. Basic Regex Functions

ðŸ”¹ re.search() â€“ Search for a pattern in a string

Returns the first occurrence of the pattern.


import re

text = "My phone number is 123-456-7890"
pattern = r"\d{3}-\d{3}-\d{4}"  # Matches a phone number format

match = re.search(pattern, text)
if match:
    print("Found:", match.group())  # Output: 123-456-7890

ðŸ”¹ re.match() â€“ Match at the start of a string

Only returns a match if the pattern appears at the **beginning** of the string.


text = "Hello World"
pattern = r"Hello"

if re.match(pattern, text):
    print("Pattern found at the beginning!")  # Output: Pattern found at the beginning!

ðŸ”¹ re.findall() â€“ Find all matches in a string

Returns a list of all occurrences of the pattern.


text = "The numbers are 42, 101, and 56."
pattern = r"\d+"  # Matches one or more digits

matches = re.findall(pattern, text)
print(matches)  # Output: ['42', '101', '56']

ðŸ”¹ re.sub() â€“ Replace text using regex

Replaces all occurrences of a pattern with a new string.


text = "The sky is blue."
pattern = r"blue"

new_text = re.sub(pattern, "red", text)
print(new_text)  # Output: The sky is red.

3. Common Regex Metacharacters

The following metacharacters are commonly used in regex patterns:

Metacharacter Description Example
. Matches any character except a newline r"a.b" matches "acb", "a1b"
\d Matches any digit (0-9) r"\d+" finds "123", "456"
\w Matches any word character (A-Z, a-z, 0-9, _) r"\w+" finds "Hello", "World"
\s Matches any whitespace (space, tab, newline) r"\s+" matches spaces
^ Matches the start of a string r"^Hello" matches "Hello World"
$ Matches the end of a string r"end$" matches "this is the end"
[] Character set, matches any one of the included characters r"[aeiou]" matches vowels

import re

text = "Contact us at support@example.com or sales@example.com"
pattern = r"\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b"

print(emails)  # Output: ['support@example.com', 'sales@example.com']

âœ… Validate a Phone Number


import re

def is_valid_phone(phone):
    pattern = r"^\d{3}-\d{3}-\d{4}$"  # Matches 123-456-7890 format
    return bool(re.match(pattern, phone))

print(is_valid_phone("123-456-7890"))  # True
print(is_valid_phone("1234567890"))    # False
