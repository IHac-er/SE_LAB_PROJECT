Python File Handling

Learn how to read, write, and manipulate files in Python using built-in functions.

What is File Handling?

Python allows handling files for reading, writing, and appending using built-in functions.

1. Reading a File

Use open() with mode "r" to read a file.


file = open("example.txt", "r")  
content = file.read()
print(content)
file.close()

2. Writing to a File

Use open() with mode "w" to write. This will overwrite existing content.


file = open("example.txt", "w")
file.write("Hello, Python!")
file.close()

3. Appending to a File

Use mode "a" to append new content without deleting existing data.


file = open("example.txt", "a")
file.write("\nAppending new line.")
file.close()

4. Reading Line by Line

Use readline() to read one line at a time.


file = open("example.txt", "r")
line = file.readline()
while line:
    print(line.strip())  # Removes extra newlines
    line = file.readline()
file.close()

5. Using With Statement

The with statement automatically closes the file.


with open("example.txt", "r") as file:
    content = file.read()
    print(content)

6. Handling File Exceptions

Handle errors using try-except blocks.


try:
    with open("nonexistent.txt", "r") as file:
        print(file.read())
except FileNotFoundError:
    print("Error: File not found.")
