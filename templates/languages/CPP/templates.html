Understanding Templates in C++

Templates provide a way to define generic functions and classes in C++, allowing code reusability and flexibility.

What are Templates?

Templates in C++ allow functions and classes to work with different data types without having to rewrite code multiple times.

There are two types of templates in C++:

  • Function Templates
  • Class Templates

Example: Function Template

#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << "Sum of integers: " << add(5, 10) << endl;
    cout << "Sum of floats: " << add(3.5, 2.1) << endl;
    return 0;
}

Explanation: The `add` function works with both integers and floating-point numbers using a single template.

Example: Class Template

#include <iostream>
using namespace std;

template <typename T>
class Box {
private:
    T value;
public:
    Box(T v) { value = v; }
    void display() { cout << "Value: " << value << endl; }
};

int main() {
    Box intBox(10);
    Box doubleBox(5.7);

    intBox.display();
    doubleBox.display();
    return 0;
}

Explanation: The `Box` class template allows storing and displaying different data types.

Advantages of Templates

  • Increases code reusability
  • Reduces redundancy
  • Works with different data types
  • Improves maintainability
