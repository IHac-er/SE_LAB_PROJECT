Understanding the Standard Template Library (STL) in C++

The STL provides reusable, efficient template classes and functions for data structures and algorithms.

Components of STL

  • Containers: Data structures like vectors, lists, and maps.
  • Algorithms: Sorting, searching, etc.
  • Iterators: Provide a way to access elements in a container.

Vectors in STL

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v = {1, 2, 3, 4, 5};
    v.push_back(6); // Add element

    for (int num : v) {
        cout << num << " ";
    }
    return 0;
}

Explanation: The vector dynamically resizes itself and provides easy insertion/deletion.

Lists in STL

#include <iostream>
#include <list>
using namespace std;

int main() {
    list<int> l = {10, 20, 30};
    l.push_front(5);
    l.push_back(40);

    for (int num : l) {
        cout << num << " ";
    }
    return 0;
}

Explanation: The list container allows bidirectional traversal and efficient insertions/deletions.

Maps in STL

#include <iostream>
#include <map>
using namespace std;

int main() {
    map<string, int> age;
    age["Alice"] = 25;
    age["Bob"] = 30;

    for (auto pair : age) {
        cout << pair.first << " - " << pair.second << endl;
    }
    return 0;
}

Explanation: The map stores key-value pairs and sorts them by key.
