Understanding Polymorphism in C++

Polymorphism allows objects of different types to be treated as objects of a common super type.

What is Polymorphism?

Polymorphism is one of the fundamental concepts of Object-Oriented Programming (OOP). It allows a single interface to represent different types or multiple implementations.

Types of Polymorphism in C++

  • Compile-time Polymorphism: Achieved using function overloading and operator overloading.
  • Run-time Polymorphism: Achieved using method overriding and virtual functions.

Example: Function Overloading (Compile-time Polymorphism)

#include <iostream>
using namespace std;

class Math {
public:
    int add(int a, int b) {
        return a + b;
    }
    double add(double a, double b) {
        return a + b;
    }
};

int main() {
    Math obj;
    cout << "Sum of integers: " << obj.add(5, 10) << endl;
    cout << "Sum of doubles: " << obj.add(3.5, 2.2) << endl;
    return 0;
}

Example: Operator Overloading

#include <iostream>
using namespace std;

class Complex {
public:
    int real, imag;
    
    Complex(int r, int i) {
        real = r;
        imag = i;
    }

    Complex operator + (const Complex& obj) {
        return Complex(real + obj.real, imag + obj.imag);
    }

    void display() {
        cout << real << " + " << imag << "i" << endl;
    }
};

int main() {
    Complex c1(2, 3), c2(4, 5);
    Complex c3 = c1 + c2;
    c3.display();
    return 0;
}

Example: Virtual Functions (Run-time Polymorphism)

#include <iostream>
using namespace std;

class Animal {
public:
    virtual void sound() {
        cout << "Animal makes a sound" << endl;
    }
};

class Dog : public Animal {
public:
    void sound() override {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Animal* a;
    Dog d;
    a = &d;
    a->sound(); // Calls Dog's overridden method
    return 0;
}

Abstract Classes and Pure Virtual Functions

When a class has at least one pure virtual function, it becomes an abstract class and cannot be instantiated.

#include <iostream>
using namespace std;

class Shape {
public:
    virtual void draw() = 0; // Pure virtual function
};

class Circle : public Shape {
public:
    void draw() override {
        cout << "Drawing a Circle" << endl;
    }
};

int main() {
    Circle c;
    c.draw();
    return 0;
}
