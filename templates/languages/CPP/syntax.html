Understanding C++ Syntax

Learn the basic structure and rules of writing C++ programs.

Basic Structure of a C++ Program

A simple C++ program follows this structure:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Key Syntax Elements

  • Header Files: `#include <iostream>` is required for input-output operations.
  • Namespace: `using namespace std;` avoids prefixing standard functions with `std::`.
  • Main Function: `int main()` is the entry point of every C++ program.
  • Output Statement: `cout << "Hello, World!";` prints text to the console.
  • Return Statement: `return 0;` signals the successful execution of the program.

Comments in C++

C++ supports two types of comments:

// This is a single-line comment

/*
   This is a multi-line comment.
   It spans multiple lines.
*/

Code Blocks and Indentation

C++ uses curly braces `{}` to define code blocks:

if (x > 0) {
    cout << "Positive number";
} else {
    cout << "Non-positive number";
}
