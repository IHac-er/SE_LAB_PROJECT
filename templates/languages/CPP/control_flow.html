Understanding Control Flow in C++

Control flow statements determine the execution order of program instructions.

Conditional Statements

C++ provides the following conditional statements:

  • if Statement: Executes a block of code if a condition is true.
  • if-else Statement: Executes one block if the condition is true, otherwise executes another block.
  • else-if Ladder: Checks multiple conditions.
  • switch Statement: Selects from multiple options.

Looping Statements

C++ supports various loops to execute code multiple times:

  • for Loop: Executes a block a specific number of times.
  • while Loop: Executes a block while a condition is true.
  • do-while Loop: Executes a block at least once, then repeats while the condition is true.

Example: if-else Statement

#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Enter a number: ";
    cin >> num;

    if (num > 0)
        cout << "Positive number" << endl;
    else if (num < 0)
        cout << "Negative number" << endl;
    else
        cout << "Zero" << endl;

    return 0;
}

Example: for Loop

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Example: while Loop

#include <iostream>
using namespace std;

int main() {
    int count = 1;
    while (count <= 5) {
        cout << "Count: " << count << endl;
        count++;
    }
    return 0;
}

Example: do-while Loop

#include <iostream>
using namespace std;

int main() {
    int count = 1;
    do {
        cout << "Count: " << count << endl;
        count++;
    } while (count <= 5);
    return 0;
}

Example: switch Statement

#include <iostream>
using namespace std;

int main() {
    int choice;
    cout << "Enter a number (1-3): ";
    cin >> choice;

    switch (choice) {
        case 1:
            cout << "You selected option 1." << endl;
            break;
        case 2:
            cout << "You selected option 2." << endl;
            break;
        case 3:
            cout << "You selected option 3." << endl;
            break;
        default:
            cout << "Invalid selection." << endl;
    }
    
    return 0;
}
