Understanding Lambda Functions in C++

Lambda functions provide a concise way to define anonymous functions inline.

What are Lambda Functions?

Lambda functions are anonymous functions (i.e., functions without a name) that can be used for short, simple tasks.

Syntax:

[capture_list](parameters) -> return_type {
    // Function body
};

Basic Example

#include <iostream>
using namespace std;

int main() {
    auto add = [](int a, int b) {
        return a + b;
    };
    cout << "Sum: " << add(5, 3) << endl;
    return 0;
}

Explanation: A lambda function is assigned to add and used to add two numbers.

Lambda with Capturing Variables

#include <iostream>
using namespace std;

int main() {
    int x = 10;
    auto printX = [x]() {
        cout << "Captured value: " << x << endl;
    };
    printX();
    return 0;
}

Explanation: The lambda function captures the variable x from the surrounding scope.

Modifying Captured Variables

#include <iostream>
using namespace std;

int main() {
    int x = 10;
    auto modifyX = [&x]() {
        x += 5;
    };
    modifyX();
    cout << "Modified value: " << x << endl;
    return 0;
}

Explanation: The lambda captures x by reference, allowing modifications.

Lambda as a Parameter

#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

void applyFunction(vector& v, auto func) {
    for (int& num : v) {
        num = func(num);
    }
}

int main() {
    vector<int> v = {1, 2, 3, 4, 5};
    applyFunction(v, [](int n) { return n * 2; });

    for (int num : v) {
        cout << num << " ";
    }
    return 0;
}

Explanation: A lambda function is passed to applyFunction to modify vector elements.

Example: Lambda with STL Algorithms

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> nums = {1, 2, 3, 4, 5};
    
    // Using lambda with for_each
    for_each(nums.begin(), nums.end(), [](int n) {
        cout << n * 2 << " ";
    });

    return 0;
}

Explanation: A lambda function is used inside for_each to process each element in the vector.

Example: Returning from a Lambda

#include <iostream>
using namespace std;

int main() {
    auto square = [](int x) -> int {
        return x * x;
    };
    
    cout << "Square of 6: " << square(6) << endl;
    return 0;
}

Explanation: The lambda function square returns the square of the input value.
