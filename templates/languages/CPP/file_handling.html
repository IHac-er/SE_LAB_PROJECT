Understanding File Handling in C++

File handling in C++ allows reading from and writing to files using streams.

What is File Handling?

File handling enables data storage and retrieval from files. C++ provides the fstream library for this purpose.

File operations include:

  • Opening a file
  • Reading from a file
  • Writing to a file
  • Closing a file

Including the fstream Library

To perform file operations, include the fstream header:

#include <fstream>
using namespace std;

Example: Writing to a File

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream file("example.txt"); // Create and open file
    file << "Hello, this is a file write operation in C++!";
    file.close(); // Close the file
    return 0;
}

Explanation: The ofstream object creates and writes to a file named example.txt.

Example: Reading from a File

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ifstream file("example.txt"); // Open file for reading
    string line;
    while (getline(file, line)) {
        cout << line << endl;
    }
    file.close(); // Close the file
    return 0;
}

Explanation: The ifstream object reads from example.txt line by line.

Appending to a File

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream file("example.txt", ios::app); // Open file in append mode
    file << "\nAppending new data to the file.";
    file.close();
    return 0;
}

Explanation: This example demonstrates how to append text to an existing file.

File Modes in C++

When opening a file, different modes can be used:

  • ios::in → Read mode
  • ios::out → Write mode
  • ios::app → Append mode
  • ios::binary → Binary mode

Example: Binary File Handling

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream outFile("data.bin", ios::binary); 
    int num = 42;
    outFile.write(reinterpret_cast(&num), sizeof(num));
    outFile.close();

    ifstream inFile("data.bin", ios::binary);
    int readNum;
    inFile.read(reinterpret_cast(&readNum), sizeof(readNum));
    inFile.close();

    cout << "Number read from file: " << readNum << endl;
    return 0;
}

Explanation: This example demonstrates writing and reading binary data to and from a file.
