Understanding Data Types in C++

C++ supports various data types to store different kinds of values.

Primary Data Types

These are the fundamental data types in C++:

Data Type Size Example
int 4 bytes int num = 10;
float 4 bytes float pi = 3.14;
double 8 bytes double precise = 3.141592;
char 1 byte char letter = 'A';
bool 1 byte bool isTrue = true;

Derived Data Types

Derived types are based on primary data types.

  • Arrays: int numbers[5] = {1, 2, 3, 4, 5};
  • Pointers: int *ptr;
  • References: int &ref = num;

User-Defined Data Types

C++ also supports user-defined types like:

  • Structures: struct Car { string brand; int year; };
  • Classes: class Car { public: string brand; int year; };
  • Enumerations: enum Color { RED, GREEN, BLUE };

Example Program

Here's a simple program demonstrating different data types:

#include <iostream>
using namespace std;

int main() {
    int num = 10;
    float pi = 3.14;
    char letter = 'A';
    bool isTrue = true;
    
    cout << "Integer: " << num << endl;
    cout << "Float: " << pi << endl;
    cout << "Character: " << letter << endl;
    cout << "Boolean: " << isTrue << endl;

    return 0;
}
