Understanding Memory Management in C++

Memory management in C++ involves dynamically allocating and deallocating memory using pointers.

Types of Memory in C++

  • Stack Memory: Stores local variables, automatically managed.
  • Heap Memory: Manually allocated memory using new and delete.

Dynamic Memory Allocation

Memory can be dynamically allocated using new and released using delete.

#include <iostream>
using namespace std;

int main() {
    int* ptr = new int; // Allocate memory
    *ptr = 10;
    cout << "Value: " << *ptr << endl;
    delete ptr; // Deallocate memory
    return 0;
}

Explanation: The new operator allocates memory for an integer, and delete releases it.

Dynamic Arrays

#include <iostream>
using namespace std;

int main() {
    int* arr = new int[5]; // Allocate array
    for (int i = 0; i < 5; i++)
        arr[i] = i * 10;
    
    for (int i = 0; i < 5; i++)
        cout << arr[i] << " ";
    
    delete[] arr; // Deallocate array
    return 0;
}

Explanation: new is used to allocate an array dynamically, and delete[] frees it.

Memory Leaks

Memory leaks occur when dynamically allocated memory is not properly deallocated.

#include <iostream>
using namespace std;

void memoryLeak() {
    int* ptr = new int(100);
    // No delete statement, causing memory leak
}

int main() {
    memoryLeak();
    return 0;
}

Prevention: Always pair new with delete to avoid memory leaks.

Example: Smart Pointers to Avoid Memory Leaks

#include <iostream>
#include <memory>
using namespace std;

int main() {
    unique_ptr<int> ptr = make_unique<int>(10);
    cout << "Value: " << *ptr << endl;
    return 0;
}

Explanation: The unique_ptr automatically deallocates memory when it goes out of scope, preventing leaks.

Example: Shared Pointers

#include <iostream>
#include <memory>
using namespace std;

int main() {
    shared_ptr<int> ptr1 = make_shared<int>(20);
    shared_ptr<int> ptr2 = ptr1; // Shared ownership

    cout << "Shared Value: " << *ptr1 << endl;
    return 0;
}

Explanation: The shared_ptr allows multiple references to the same memory, automatically managing deallocation.
