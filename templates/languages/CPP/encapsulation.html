Understanding Encapsulation in C++

Encapsulation is one of the core principles of Object-Oriented Programming. It restricts direct access to certain details of an object and allows controlled access through methods.

What is Encapsulation?

Encapsulation is the bundling of data (variables) and methods (functions) that operate on the data into a single unit, i.e., a class. It helps protect data from unintended access and modification.

Encapsulation Example

In the example below, data members of the class are private and can only be accessed through public methods.

#include <iostream>
using namespace std;

class BankAccount {
private:
    double balance;

public:
    // Constructor
    BankAccount(double initialBalance) {
        balance = initialBalance;
    }

    // Setter function
    void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
        }
    }

    // Getter function
    double getBalance() {
        return balance;
    }
};

int main() {
    BankAccount account(1000);  // Create an account with $1000
    account.deposit(500);  // Deposit $500
    cout << "Current balance: $" << account.getBalance() << endl;

    return 0;
}

Why Use Encapsulation?

  • Data Hiding: Prevents direct access to sensitive data.
  • Improved Maintainability: Changes can be made without affecting other parts of the program.
  • Code Reusability: Encapsulated code can be reused across different applications.

Access Modifiers in Encapsulation

Encapsulation in C++ is implemented using access specifiers:

  • private: Members are accessible only within the same class.
  • public: Members are accessible from outside the class.
  • protected: Members are accessible in derived classes.

Encapsulation with Getters and Setters

Getters and setters allow controlled access to private variables.

#include <iostream>
using namespace std;

class Student {
private:
    string name;

public:
    // Setter method
    void setName(string studentName) {
        name = studentName;
    }

    // Getter method
    string getName() {
        return name;
    }
};

int main() {
    Student student;
    student.setName("Alice");
    cout << "Student Name: " << student.getName() << endl;

    return 0;
}
